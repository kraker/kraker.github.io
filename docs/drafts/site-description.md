This string renders as the default `<meta name="description">`, the default OG / Twitter card description, and (on Hextra) is surfaced in search engine snippets and social previews sitewide. So it functions as the one-line elevator pitch.

## Claude-drafted alternative (parked)

Tradeoffs vs. the current line:

- Probably better for the consulting audience; arguably worse for readers who came for the person, not the practice.
